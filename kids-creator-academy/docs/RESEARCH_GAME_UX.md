# Research notes — Game + UX for AI creative learning (ages 8–11)

> Research before implementation. Not a certification. Patterns only.

## Official / authoritative (international)

| Source | Takeaway for this product |
|--------|---------------------------|
| Nielsen Norman Group — Children's UX / Kids cognition | Narrow age band; clear goal + how; one primary action; large targets; short text |
| WCAG 2.2 | Contrast, focus, keyboard alternatives to drag-drop |
| UNICEF — AI and Children | Best interests of child; safety, privacy, agency |
| UNESCO — GenAI in education | Human agency; AI literacy; adult oversight |
| ICO Children's Code / COPPA | High privacy default; minimize data; no pressuring kids to share |
| Common Sense Education — AI literacy | Age-fit lessons; critical thinking about AI mistakes |

## Successful global products (patterns, not clones)

| Product | Useful pattern | Do NOT copy |
|---------|----------------|-------------|
| **Scratch** | Project-first, create → share with safety, blocks as scaffold | Mascot/IP, open social feed |
| **Code.org** | Course → lesson → level path; clear next level | Brand assets |
| **Duolingo** | Short levels, XP, encouragement, progress path | Aggressive streaks / FOMO for under-13 |
| **Kahoot** | Quiz as game, short rounds, celebration | Public shaming last place |
| **Prodigy / ABCmouse** | Map/quest metaphor, rewards as unlocks | Pay-to-win patterns |
| **Minecraft Education** | Creative sandbox with goals | Complex UI for first session |

## Gamification for 8–11 (evidence-aligned)

**Keep**
- XP and stars as **feedback**, not punishment
- Unlock next quest by completing challenge (quiz / drag puzzle)
- Cheerful copy: “Thử cách khác”, never “Thất bại / thua”
- Collection book (badges, assets) as main reward
- Optional **class star board** (nicknames only, teacher view) — not global public web ranking

**Avoid (spec + child safety)**
- Global public leaderboard “toàn hệ thống”
- “Bạn đang thua bạn khác”
- Loot box / pay-to-win
- 24h streak loss pressure
- Real-name ranking

## UI layout for kids

- Single main column for practice screens (no competing middle sidebar)
- Nav: max 4–5 items, labels + icons
- Game board: large drop zones, drag + click-to-place
- Colorful but soft clay surfaces — not neon “AI SaaS plastic”
